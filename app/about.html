About the CIDR Calculator Project

A web application deployed on Microsoft Azure

Calculator About

Project Overview

This CIDR Calculator web application is designed as a test project for deploying web applications to Microsoft Azure cloud platform. The project demonstrates fundamental concepts of web development and cloud deployment strategies.

The application provides a simple but powerful tool for network administrators and IT professionals to quickly calculate network information based on CIDR notation, including subnet masks, network addresses, broadcast addresses, and usable IP ranges.

Azure Deployment

This project explores the workflow of deploying a web application to Microsoft Azure using the following services:

Azure App Service
Azure DevOps
GitHub Actions
Application Insights

The deployment process includes:

  1. Setting up a GitHub repository for version control
  2. Configuring continuous integration and continuous deployment (CI/CD) pipelines
  3. Provisioning Azure resources using Infrastructure as Code (IaC)
  4. Implementing monitoring and logging for application performance
  5. Setting up staging and production environments

CIDR Calculator Features

The CIDR (Classless Inter-Domain Routing) calculator implements the following features:

  • Conversion of CIDR notation to subnet mask
  • Calculation of network address
  • Calculation of broadcast address
  • Determination of first and last usable host addresses
  • Calculation of total number of usable hosts
  • Input validation for proper IP address and prefix formats

This tool is particularly useful for network planning, troubleshooting, and documentation.

Technologies Used

The application is built using:

  • Frontend: HTML5, CSS3, and vanilla JavaScript
  • Version Control: Git and GitHub
  • Cloud Platform: Microsoft Azure
  • DevOps: Azure DevOps or GitHub Actions
  • Monitoring: Azure Application Insights

The project deliberately uses minimal dependencies to focus on the deployment process rather than complex application architecture.

Learning Objectives

This project serves as a learning platform for:

  • Understanding the Azure cloud ecosystem
  • Implementing modern DevOps practices
  • Setting up efficient CI/CD pipelines
  • Deploying and scaling web applications in the cloud
  • Implementing proper monitoring and logging
  • Applying infrastructure as code principles
